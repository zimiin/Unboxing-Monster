# UNBOXING APP
Public RNG API를 활용한 공정한 랜덤박스 어플입니다.

### 사용 기술
React 17.0.1   
React Native 0.64.1   

### 실행 방법
프로젝트 클론 후, 아래의 명령어를 실행한다.
~~~
npm install   
cd ios && pod install   
~~~
